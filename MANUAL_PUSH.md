# GitHub-ga Yuklash (Qo'llanma)

Siz Git-ni o'rnatgansiz, lekin VS Code uni hali ko'rmayapti.
Iltimos, **VS Code dasturini butunlay yopib, qayta oching**.

Keyin shu papkada terminal ochib (Terminal -> New Terminal), quyidagi buyruqlarni yozing:

```bash
git remote set-url origin https://github.com/link-edition/linkedition.git
git add .
git commit -m "Site Update"
git push -u origin main
```

Agar xato bersa, demak kompyuterni "Restart" qilish kerak.
